# Background Music

Letakkan file musik background dengan nama `background.mp3` di folder ini.

## Rekomendasi:
- Format: MP3
- Durasi: 2-5 menit (akan di-loop)
- Genre: Lo-fi, acoustic, atau musik instrumental yang ringan
- Ukuran: Maksimal 5MB untuk loading cepat

## Sumber musik gratis:
- pixabay.com/music
- freemusicarchive.org
- bensound.com (dengan atribusi)

## Catatan:
Jika file musik tidak ada, tombol play tetap muncul tapi tidak ada suara.
